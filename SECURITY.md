# Security Policy

## Supported Versions

AgenticFlowX is currently in **alpha**. There is no formal support window — we patch on a best-effort basis.

## Reporting a Vulnerability

Please open a [GitHub issue](https://github.com/AgenticFlowX/agenticflowx/issues) or [discussion](https://github.com/AgenticFlowX/agenticflowx/discussions) with:

- A short summary of the issue
- Steps to reproduce or a proof of concept
- Any logs, stack traces, or screenshots that help illustrate the problem

We will review reports as quickly as we can. While a fix is in progress, please avoid publicly disclosing exploit details.

Thank you for helping keep AgenticFlowX users safe.
